OpenTOSCA Vintner Shop Demo is a simple Node.js application demonstrating a component connecting to a database.

## Getting Started 

Install dependencies.

```
npm install --frozen
```

Start using IPv4.

```
PORT=3000 INTERFACE=0.0.0.0 npm start
```

Start using IPv6.

```
PORT=3000 INTERFACE=:: npm start
```
